Personal projects

Osasuna FC

The Osasuna FC website is a dynamic web app showcasing the renowned football club, Osasuna, and delivering real-time updates to fans. Developed with Node.js, Express, EJS, Tailwind CSS, and MongoDB, it exemplifies my expertise in full-stack web development.

ShutterSpot

Shutterspot is a cutting-edge application designed for booking photographers, providing users with a seamless and intuitive platform to connect with talented professionals. Tech Stack: React with Vite for frontend development, MongoDB for database management, Node.js for server-side operations.

SkillUp

SkillUp is a platform that helps users discover in-demand skills and find relevant learning resources. It employs K-means clustering for demand analysis, with Node.js and Flask for backend and React Native for frontend development. Tech Stack: K-means clustering, Node.js, Flask, React Native Highlights: SkillUp simplifies skill discovery and learning by analyzing demand levels and directing users to appropriate resources.

UniqueWear

UniqueWear is a Node.js-based e-commerce platform offering a diverse selection of clothing, shoes, and jewelry for men and women. Tech Stack: Node.js, Express, EJS, Tailwind CSS, MongoDB Highlights: With its robust tech stack, UniqueWear delivers an intuitive interface and efficient performance, catering to fashion enthusiasts with its extensive product range.
